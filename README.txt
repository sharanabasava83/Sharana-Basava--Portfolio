Sharanabasava - Portfolio Website
==================================

Files in this folder:
- index.html               -> the portfolio website (open this in any browser)
- profile.jpg               -> your profile photo (used by index.html)
- Sharanabasava_Resume.pdf  -> your resume (linked via the "Download Resume" button)

IMPORTANT: Keep all 3 files together in the same folder.
The website links to profile.jpg and the resume PDF by filename,
so if you separate or rename them, those parts will break.

HOW TO VIEW IT NOW:
Just double-click index.html and it opens in your browser.

HOW TO PUT IT ONLINE (so you can share a link with recruiters):
Easiest free option - GitHub Pages:
1. Create a free GitHub account at github.com (if you don't have one)
2. Create a new repository (e.g. "portfolio")
3. Upload all 3 files from this folder into that repository
4. Go to Settings -> Pages -> set source to "main" branch, root folder
5. GitHub will give you a live link like:
   https://yourusername.github.io/portfolio/
6. Share that link on your resume, LinkedIn, and with recruiters

Other easy free options: Netlify Drop (netlify.com/drop - just drag
this folder in) or Render (render.com - static site).
