## 🎯 Quick Start

### 1️⃣ Navigate to Project
```bash
cd Sharanabasava-Portfolio
```

### 2️⃣ Install Dependencies
```bash
npm install
```
This downloads all required packages (~500MB)

### 3️⃣ Start Development
```bash
npm run dev
```
Opens at http://localhost:3000 automatically

### 4️⃣ Customize Your Content
Edit files in `src/components/` to add your information

### 5️⃣ Deploy
```bash
npm run build
npm run preview  # Test production build
```
